## 💻 Linguagens de Programação

Linguagens de programação são conjuntos de regras e sintaxe que permitem escrever códigos que o computador entende.

🚀 Funcionam como uma ponte entre humanos e máquinas.

### Importante saber
O computador entende apenas 0 e 1 (binário)

Linguagens facilitam a comunicação

Permitem criar: Sites,Jogos, Aplicativos 

## 📊 Níveis de Linguagem

Nível	Exemplos

Baixo nível	Binário, Assembly

Médio	C, C++

Alto nível	Java, Python, JavaScript


- Quanto maior o nível, mais fácil para humanos

🔍 Estrutura:

const → tipo de declaração

novaVariavel → nome

= → atribuição

200 → valor

## 🔑 Tipos de declaração
🔄 let

let nome = "Maria";

nome = "Joana";

👉 Pode mudar

🔒 const

const pi = 3.14;

👉 Não pode mudar

🐫 camelCase

📖 O que é?

Forma padrão de escrever variáveis:

Começa com letra minúscula

Próximas palavras com maiúscula

Sem espaços

✅ Exemplos

nomeCompleto

numeroDeAlunos

dataDeNascimento

mediaFinal

🎯 Por que usar?

Padrão do JavaScript
Facilita leitura
Código mais profissional

## Juntando caracteres

🔗 Concatenação
📖 O que é?

Juntar textos + variáveis

🧪 Exemplo 
let nome = "Robersvaldo"

let idade = 22;

console.log("Olá, meu nome é " + nome + " e eu tenho " + idade + " anos");
✨ Template Strings (moderno)
let nome = "João";
let idade = 25;

🔗 Template String

console.log(`Olá, meu nome é ${nome} e eu tenho ${idade} anos`);
