LINGUAGENS DE PROGRAMAÇÃO
Linguagens de programação são conjuntos de regras e
sintaxe que permitem aos programadores escrever
códigos que podem ser entendidos pelos computadores.
Elas funcionam como uma ponte entre as ideias humanas
e a máquina.
LINGUAGENS DE PROGRAMAÇÃO
Essencialmente, uma linguagem de programação fornece
uma maneira de instruir o computador a fazer tarefas
específicas, seja para criar sites, jogos, aplicativos, sistemas
de inteligência artificial ou qualquer outro tipo de software.
LINGUAGENS DE PROGRAMAÇÃO
O computador, por sua vez, entende apenas linguagens
de máquina (sequências de 0s e 1s). Mas como seria
impraticável escrever diretamente em código binário,
criamos linguagens de programação para facilitar esse
processo.

Declaração de Variáveis em JavaScript
🔹 O que significa declarar uma variável?
Declarar uma variável é informar ao programa que você deseja
reservar um espaço na memória para armazenar um determinado
valor.
Essa variável poderá ser usada, modificada ou consultada ao longo
do código.
Declaração de Variáveis em JavaScript
Palavras-chave para declaração
✅ let
Usada para declarar variáveis que podem ter seus valores alterados.
VARIÁVEIS EM JS

Declaração de Variáveis em JavaScript
Palavras-chave para declaração
✅ const
Usada para declarar variáveis cujo valor não deve ser modificado depois de
atribuído.
VARIÁVEIS EM JS
a variável pi recebe 3.14 como foi usado
const ela NÃO pode receber outro valor!
Ela é constante!
🟨 O que é camelCase?
camelCase é uma convenção de escrita de nomes de variáveis e funções onde:
A primeira palavra começa com letra minúscula
As demais palavras começam com letra maiúscula
Não há espaços, acentos ou caracteres especiais
Essa forma de escrita se parece com as “corcovas de um camelo” 🐫 — por isso o
nome camelCase.

As template strings foram introduzidas no ES6 e tornam a vida muito mais fácil.
Sintaxe:
Use crases ` em vez de aspas simples ou duplas.
Insira variáveis usando ${variavel}.
